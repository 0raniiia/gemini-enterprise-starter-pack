# Get Started with Gemini Enterprise

> Full slide deck content: see `GE_GetStarted_Deck_EN.md` in the root `docs/` folder.

## Your first 3 actions (do these now)

**1. Try Gemini in Gmail (5 min)**
Open any email thread → click ✨ → ask Gemini to draft a reply. That's your first wow moment.

**2. Summarize a long document with Gemini in Docs (5 min)**
Open any document over 5 pages → click ✨ → ask: *"Summarize this in 5 key bullet points."*

**3. Build a knowledge base in NotebookLM (10 min)**
Go to [notebooklm.google.com](https://notebooklm.google.com) → upload 2–3 internal docs → ask a question in plain language.

## Security & Privacy quick facts
- ✅ Your conversations are never used to train Google's models
- ✅ Data stays within your Google Workspace environment

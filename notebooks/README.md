## Quick start
1. Click any `.ipynb` file → click "Open in Colab" button
2. Go to the Secrets panel (🔑) → add `GEMINI_API_KEY`
3. Run all cells (`Runtime → Run all`)

- **Gemini API Key** (prototyping): get yours at [aistudio.google.com](https://aistudio.google.com)
